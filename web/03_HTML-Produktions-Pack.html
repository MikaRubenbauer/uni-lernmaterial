Erstellen H1 — Minimales HTML-Dokument („Hello World")

Quelltext (minimal & valide HTML5)Ausgabe
<!DOCTYPE html> <html lang="de"> <head> <meta charset="UTF-8"> <title>Titel</title> </head> <body>Hello World</body> </html>
Hello World
Pflicht-Bausteine: <!DOCTYPE html> (HTML5-Deklaration, KEIN Tag), <html> (Root), <head> mit <title> (für Validität nötig), <body> mit dem sichtbaren Inhalt. <meta charset> + lang = empfohlen.

Nennen H4 — Zwei Tags im <head> + Wirkung

TagWirkung
<title>Titel im Browser-Tab / Lesezeichen / Suchergebnis. Pflicht für valide Seite.
<meta charset="UTF-8">Legt die Zeichenkodierung fest → Umlaute korrekt.
<link rel="stylesheet" href>Bindet eine externe CSS-Datei ein.
<script src>Bindet JavaScript ein.
<meta name="viewport">Steuert Skalierung auf Mobilgeräten.

In der Klausur reichen zwei mit jeweils einem Satz Wirkung. Sichere Wahl: <title> + <link rel="stylesheet">.

Erstellen H5 — Hyperlink & URL-Bestandteile

HyperlinkAusgabe

Beschreibung

Ein Hyperlink trägt zwei Bestandteile: das Verweisziel (URL, href) und die textuelle Beschreibung (Linktext).

URL-BestandteilBeispiel
Protokollhttps://
Host (Subdomain.Domain.TLD)www.example.com
Port (optional):8080
Pfad/ordner/datei.html
Query (optional)?param=wert
Fragment (optional)#abschnitt
Klausur-Frage „ein Bestandteil nennen + Beispiel": z.B. Protokoll https, oder Host example.com. Ein Beispiel genügt.
